## üéØ Overview

**Infiniology** is a comprehensive philosophical framework that seeks to understand the fundamental structure of reality through a central principle: the **Law of the Limitation of Infinity**. Originally developed as a tool for integrating mythology within fictional worlds, this framework has evolved into a sophisticated system that bridges science, philosophy, spirituality, and consciousness studies.

### Key Thesis
> "The infinite must be limited to be knowable."

This foundational principle posits that absolute infinity is imperceptible and can only be experienced through limitation, which gives rise to the dualistic universe we inhabit.

## üß† Core Principles

### 1. The Law of the Limitation of Infinity
The infinite, absolute reality (the Source) is imperceptible and can only be known through a process of limitation, which gives rise to the dualistic universe we experience.

### 2. Vibration as Perception
**Vibration is the perceptible form of the duality of harmony and disharmony.** This breakthrough realization provides a tangible, measurable link between abstract cosmic laws and sensory experience.

### 3. The Absolute Principles: Source and Self
- **Source**: The ultimate, undifferentiated, infinite reality
- **Self**: The principle of individuation, the "I AM" that allows for unique identity

### 4. The Active Observer
The Self is not a passive recipient but an active participant in creating reality through perception and choice.

## üöÄ Installation & Usage

### Prerequisites
- Modern web browser
- Local web server (optional)

### Quick Start

1. **Clone or Download**
   ```bash
   git clone [repository-url]
   cd infiniology-framework
   ```

2. **Local Development**
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Using PHP
   php -S localhost:8000
   ```

3. **Open in Browser**
   Navigate to `http://localhost:8000`

### Deployment
The framework is a static HTML site and can be deployed to any web hosting service:
- GitHub Pages
- Netlify
- Vercel
- Traditional web hosting
